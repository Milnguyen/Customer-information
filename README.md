# Customer-information

## 📌 Giới thiệu
Dự án đơn giản sử dụng **JSP + JSTL + Java** để hiển thị danh sách khách hàng dưới dạng bảng.

## 🛠️ Công nghệ sử dụng
- Java
- JSP (Java Server Pages)
- JSTL (JSP Standard Tag Library)
- Apache Tomcat
- ## 📄 Mô tả chức năng
- Hiển thị danh sách khách hàng gồm:
  - Tên
  - Ngày sinh
  - Địa chỉ
  - Ảnh
- Sử dụng JSTL `<c:forEach>` để lặp dữ liệu
